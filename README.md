# Java-coding-puzzle-11

	Given a string, return a string where for every char in the original, append another. 


	 EXPECTATIONS:
		repeatChar("The")       --- "TThhee"
		repeatChar("AAbb")      --- "AAAAbbbb"
		repeatChar("Hi-There")  --- "HHii--TThheerree"
